# 🏀 Chicago Bulls Analytics Dashboard

A modern, data-driven dashboard for analyzing the performance of the Chicago Bulls. This project provides insights into team efficiency, player performance, and game trends using real-time and historical NBA data.

## 📊 Features

* 📈 Team performance metrics (Offensive/Defensive Rating, Net Rating)
* 🔍 Advanced analytics breakdown
* 📅 Game tracking (previous, current, upcoming)
* 🚑 Injury reports (if available)
* 📉 Visualized trends and efficiency metrics
* ⚡ Fast, responsive UI

---

## 🛠️ Tech Stack

**Frontend:**

* React
* Tailwind CSS
* Chart.js / Recharts

**Backend / Data Processing:**

* Python
* Pandas
* NBA API / Web scraping (BeautifulSoup)

**Deployment:**

* Vercel (Frontend)
* Render (Backend)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/rtjohnson0/ChicagoBullsDashboard.git
cd ChicagoBullsDashboard
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```
